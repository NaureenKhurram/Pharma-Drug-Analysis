# Pharma-Drug-Analysis
This project aims to visualise the prescription volume and cost of selected drugs for Cancer, Mental Health, Stroke and across NHS regions in England using Power BI


## Drugs studied by condition

### Condition  - Drug Name ###	

Stroke	- Clopidogrel	(commonly prescribed for stroke prevention)

Mental Health - Sertraline	(highly prescribed across the UK)

Cancer - Calcutamide	(widely prescribed for Prostrate Cancer)
